# AI Model Inventory (ModelHub)

Hey! This is a project I put together to help track and manage AI models. I built this because I wanted a centralized place where users could browse models, filter them by framework, and keep track of their own collection.

## 🚀 Live Demo
Check it out here: https://ai-model-inventory-d4yf.vercel.app

## 🛠 What's under the hood?
I used the MERN stack for this. 
* **Frontend:** React, Vite, and Tailwind CSS (with DaisyUI for the UI components).
* **Backend:** Node.js and Express.
* **Database:** MongoDB Atlas (for storage).
* **Auth:** JWT to keep user sessions secure.

* `/server`: The Express backend, API routes, and database models.
